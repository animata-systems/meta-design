# Meta-Design

Это пространство размышления об архитектуре, а не её фиксация. Здесь живут черновики системных взглядов, проекции, попытки разложить сложное на понятные части, брейнсторминг, сомнения и поиск формы. Нормативные решения рождаются позже и уезжают в спецификации; сюда же возвращаются контекст, обоснования и те вопросы, которые ещё не готовы стать ответами. Архитектура системы — это не одномоментный документ, а длинный разговор с самим собой, и этот репозиторий хранит именно такой разговор.
