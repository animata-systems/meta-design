## Главные выводы одной выжимкой

**Corazon Network — самостоятельная открытая сеть**, к которой может подключиться любой хост, на любом рантайме. COIL — не обязательное условие входа, а слой доверия, повышающий прозрачность и аудитабельность участника.

**Хост — это кошелёк.** Интерфейс к сети, управляющий ключами и подписывающий транзакции. Хостов может быть много: для COIL-агентов, плагины для сторонних фреймворков, кастомные реализации.

**Cora OS — это iOS + кошелёк.** Снаружи — хост, подписывающий транзакции от имени людей. Внутри — платформа с приложениями, App Store и Designer. Платформа, где создание приложения — операция внутри приложения.

**SymbioRank — встроенная в сеть репутация.** Четыре оси (реакция, резонанс, симбиоз, надёжность), детерминированно вычисляемые нодами из header-лога. Рейтинг верифицируем, портируем и относителен к горизонту наблюдения ноды.

## Связь с предыдущими сессиями

Сессия развивает и частично пересматривает выводы `2026-04-09-strategy-synthesis`. Конкретные сдвиги:

- **A2** (Cora OS = iOS для COIL-приложений) → уточнено: Cora OS = iOS + кошелёк для участников Corazon. Приложения не обязаны быть на COIL.
- **A3** (Corazon Host = независимый runtime) → обобщено: «хост» — generic-понятие, любой клиент сети. Corazon Host для COIL-агентов — частный случай.
- **F1** (трёхслойный бренд) → изменена природа: не три слоя стека, а три сущности разной природы (язык доверия, открытая сеть, flagship-платформа).
- **R6** (декомпозиция Cora OS) → снимается: Cora OS — хост + платформа, а не операционная система.

Всё остальное из 17 пунктов консолидации остаётся без изменений и усиливается.

## Статус

**Рабочий снимок понимания на 2026-04-10.** Ничего из этого не является нормой. Архитектурный сдвиг (открытая сеть, хост-кошелёк) ощущается устойчивым; SymbioRank — на стадии ранней рамки, требует проработки деталей (нормализация, sybil-защита, граница протокол/приложение).
