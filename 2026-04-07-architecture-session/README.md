# Архитектурная сессия — 2026-04-07

Запись обсуждения общей архитектурной картины системы: её компонентов, ролей действующих лиц, слоя инструментов и экономической модели, которая из всего этого вытекает. Не лог, а реконструкция — разложена по темам.

Сессия началась с проработки модели доверия (вопрос «кто видит plaintext сообщений»), но быстро ушла глубже: пользователь несколько раз вносил уточнения, которые сдвигали понимание системы. Итог — собранная картина целой системы, экономика и список открытых вопросов.

## Как читать

Если сначала хочется понять **что это за система** — `01` → `02` → `03`.

Если хочется понять **почему она может жить** — `04` → `05`.

Если хочется понять **какие решения ещё не приняты** — `open-questions.md`.

Статьи в `articles/` — это публичная проекция последних двух выводов сессии. Они не содержат внутренних решений, номеров историй и процесса — только идеи.
